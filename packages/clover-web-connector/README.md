# @web3-wallets-kit/clover-web-connector

## Installation

`npm install @web3-wallets-kit/clover-web-connector`

## Example

```typescript
import { CloverWebConnector } from '@web3-wallets-kit/clover-web-connector';

const connector = new CloverWebConnector({
  initOptions: {
    network: {
      chainId: '0x01',
    }
  }
});
```
